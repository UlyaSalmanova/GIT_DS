### Описание проекта  
Угадать загаданное компьютером число меньше, чем за 20 попыток.  
:top: [к Оглавлению](#Oглавление)  

### Какой кейс решаем?  
Нужно написать программу, которая угадывает число за минимальное число попыток.  

**Условия соревнования:**  
:small_orange_diamond: Компьютер загадывает целое число от 0 до 100, и нам его нужно угадать. Под «угадать», подразумевается «написать программу, которая угадывает число».  
:small_orange_diamond: Алгоритм учитывает информацию о том, больше ли случайное число или меньше нужного нам.  
:small_orange_diamond: Необходимо добиться того, чтобы программа угадывала число меньше, чем за 20 попыток.  

**Метрика качества**  
Результаты оцениваются по среднему количеству попыток при 1000 повторений. Необходимо добиться минимального количества попыток.  

**Что практикуем**
